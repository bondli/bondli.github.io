快速开始

3个命令搭建一个H5项目.

1. 安装系统必备

sudo npm install -g yo grunt-cli bower(前提是安装了node.js)

2. 安装脚手架

sudo npm install -g generator-admix

3. 建立项目

yo admix(需要自己先创建项目目录，然后进入该目录执行命令)

新建的项目中包含了什么？

{% highlight bash %} project_name/ ├── app/ │ ├── bower_components/ │ ├── mods/ │ └── pages/ ├── build/ ├── htmls/ └── htmls-dist/ {% endhighlight %}

新建页面

yo admix:page xxx 你将在htmls目录中发现多了一个xxx.html，同时在app/pages/下面多了一个xxx的目录

新建模块

yo admix:mods xxx 你将在app/mods/目录下发现多了一个xxx.js，这个是整个项目可以共用的模块，如果要在某个页面下建立模块，可以在前面的命令基础上增加：空格+页面名称.

启动项目

grunt 你将看到浏览器自动为你打开了项目，你点击页面进入即可.

项目打包

grunt build 打包后即可提交gitlab.
